# 📊 Sales & Operations MIS Dashboard

## Overview
A complete Sales & Operations MIS reporting project built in 
Excel / Google Sheets analyzing 200 rows of sales data.

## 📈 Key Metrics
- **Total Revenue:** ₹30.49M
- **Total Units Sold:** 1,591
- **Achievement Rate:** 57.5%
- **Top Region:** Mumbai

## 🛠️ Skills & Formulas Used
- `IF` – Status column (ACHIEVED / NOT ACHIEVED)
- `VLOOKUP` – Product revenue lookup
- `XLOOKUP` – Employee region lookup
- `SUMIF` – Region-wise total revenue
- **Pivot Tables** – Region, Employee, Product, Status summaries
- **Conditional Formatting** – Green/Red status highlighting
- **Charts** – Bar charts, Pie chart
